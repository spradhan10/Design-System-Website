Display & Position

The position propety allows you to specify the posiion of an element.
the type of position are:

position definition example
relative

When set to relative, an element's position is relative to its default position on the page

 p{
position: relative;
}
absolute

When set to absolute, an element's position is relative to its closest positioned parent element. It can be pinned to any part of the web page, but the element will still move with the rest of the doucment when the page is scrolled.

 p{
position: absolute;
width: 100%
}
fixed

When set to fixed, an element's position can be pinned to any pat of the web page. The element will remain in view no matter what.

nav{
position: fixed;
}
sticky

When set to sticky, an element can stick to a defined offset position when the user scrolls its parent container.

 .question{
text-align: center;
position: sticky;
top: 40px;
}
z-index The z-index of an element specifies how far back or how far forward an element appers on the page when it overlaps other elements. .blue-box {,
background-color: blue;
position: relative;
z-index: 1;
}
.green-box {
background-color: green;
position: relative;
top: -170px;
left: 170px;
}
display The display propety allows you to control how an element flows vertically and horizontally a document. container1 {
display: block;
}
//block elements take up as the width of their container and can have manually adjusted heights.

.container2 {
display: inline;
}
//inline elements take up as little apace as possible, and they cannot have manually adjusted width or height.

.container3 {
display: inline-block;
}
//inline-block elements can have set width and height, but they can also appear next each other and do not take up their entire container width.
float The float property can move elements as far left or as far right as possible on a web page. p{
float: right;
}
clear You can clear an element's left or right side (or both) using the clear property. .element {
clear: left;
}

.element {
clear: right;
}

.element {
clear: both;
}

.element {
clear: none;
}
